# Extract rejection decisions from BMA results

Extract rejection decisions from BMA results

## Usage

``` r
# S3 method for class 'bma_result'
extract_rejections(result, ...)
```

## Arguments

- result:

  A bma_result object from BMA analysis

- ...:

  Additional arguments (for S3 consistency)

## Value

Logical vector indicating which baskets reject the null hypothesis
