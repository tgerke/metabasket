# Extract rejection decisions from Simon analysis

Extract rejection decisions from Simon analysis

## Usage

``` r
# S3 method for class 'simon_result'
extract_rejections(result, ...)
```

## Arguments

- result:

  A `simon_result` object from `analyze_basket.simon()`

- ...:

  Additional arguments (not currently used)

## Value

Logical vector indicating which baskets were rejected (declared active)
